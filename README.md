# 新建一个react app

```jsx
npx create-react-app my-app
cd my-app
npm start
```

# Hello World
* public
  * index.html
* src
  * index.js
  * reportWebVitals.js
* .gitignore //git忽略文件
* package.json // webpack配置
* package-lock.json

***

1. ``index.html`` //html入口
2. ``index.js`` //js入口
3. ``reportWebVitals.js`` //热加载,监控相关
4. ``.gitignore`` //git忽略文件
5. ``package.json`` // 记录当前项目所依赖模块的版本信息
6. ``package-lock.json`` // 记录了node_modules目录下所有模块的具体来源和版本号以及其他的信息
