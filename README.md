## 1 声明式

* 传统命令式编程
```js
//1 定义标签
<btn>内容</btn>

//2 找到标签, 让标签渲染新的内容
data=fetch()
findBtn().setValue(data)
```

* 声明式

```js
//1 定义标签, 绑定数据变量
data="内容"
<btn>{data}</btn>

//2 变更数据
data=fetch()
```
