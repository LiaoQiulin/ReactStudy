# 4 react 介绍

react是一个构建用户界面的javascript库

## 4.1 声明式

* 传统命令式编程
```js
//1 定义标签
<btn>内容</btn>

//2 找到标签, 标签内容
data=fetch()
btn.setValue(data)
```

* 声明式

```js
//1 定时数据变量
data="内容"
//2 定义标签, 绑定数据
<btn>{data}</btn>
//3 变更数据
data=fetch()
```
